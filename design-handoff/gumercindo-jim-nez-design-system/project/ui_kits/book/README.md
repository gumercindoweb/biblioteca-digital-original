# Book Landing UI Kit — "Hay que salir del bar"

Luxury-editorial sales page for Gumercindo's introspection book, rebuilt on the design system (replaces the original midnight-navy + purple template).

## Files
- `index.html` — full landing page.
- `sections.jsx` — `Nav`, `Hero`, `Inside`, `Offer`, `Proof`, `FAQ`, `Footer`, plus two kit-local pieces: `BookCover` (the title typeset on a deep-green cover — no cover-art asset was provided) and `Seal` (gold 30-day guarantee medallion).

## Notes
- The book cover and guarantee seal are recreated with CSS/type, not imagery. Swap `BookCover` for the real cover render when available.
- Mirrors the real sales-page structure (hook → agitation → what's inside → offer + bonuses → guarantee → proof → FAQ) but in the brand's deep-green/gold luxury system.
