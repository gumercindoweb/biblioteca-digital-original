Section marker inside a workbook/diario, e.g. the five blocks of "Hay que salir del bar".

```jsx
<BlockLabel glyph="✦">Bloque 1 — La esencia</BlockLabel>
<BlockLabel glyph="◎">Bloque 2 — El espejo</BlockLabel>
```

Glyph set used across the products: ✦ esencia · ◎ espejo · ◈ ejercicio · ✍ compromiso · ▣ termómetro. Honors `--editorial-band` so it turns violet inside `[data-line="quejica"]`.
