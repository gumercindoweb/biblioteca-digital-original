The "Preguntas frecuentes" pattern from the site.

```jsx
<Accordion
  defaultOpen={[0]}
  items={[
    { q: '¿Qué incluye exactamente la consultoría?', a: 'Diagnóstico, implementación y formación…' },
    { q: '¿Cuánto tiempo toma ver resultados?', a: '…' },
  ]}
/>
```

`allowMultiple` lets several stay open. `tone="dark"` for deep-green sections.
