Editorial surface container for services, testimonials, stats.

```jsx
<Card tone="paper" interactive accentRule>
  <Eyebrow>Servicio 01</Eyebrow>
  <h3>Automatización inteligente</h3>
</Card>
<Card tone="dark" padding="xl">…</Card>
```

Tones: `paper` (default), `sunken`, `tint` (faint green), `dark` (deep-green section). `interactive` lifts on hover; `accentRule` draws a gold top rule.
