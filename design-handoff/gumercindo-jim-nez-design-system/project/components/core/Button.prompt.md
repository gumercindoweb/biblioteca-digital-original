Primary action control for the Gumercindo Jiménez brand — uppercase tracked-caps label, deep-green by default, used for CTAs like "Solicitar una auditoría".

```jsx
<Button variant="solid" size="lg" onClick={submit}>Solicitar una auditoría</Button>
<Button variant="outline" iconRight="→">Ver servicios</Button>
<Button variant="link">Seguir leyendo</Button>
```

Variants: `solid` (brand green, default CTA), `accent` (warm clay — use sparingly for the single hottest action), `outline` (gold hairline, secondary), `ghost` (tertiary), `link` (inline editorial underline). Sizes `sm | md | lg`. Supports `iconLeft` / `iconRight`, `fullWidth`, `disabled`.
