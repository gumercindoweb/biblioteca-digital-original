Small pill for service categories or metadata, e.g. "AUTOMATIZACIÓN", "VENTAS".

```jsx
<Tag tone="brand">Estrategia</Tag>
<Tag tone="gold" variant="outline">Premium</Tag>
<Tag tone="clay" variant="solid">Plazas limitadas</Tag>
```

Variants `soft | solid | outline`; tones `brand | clay | gold | neutral`.
